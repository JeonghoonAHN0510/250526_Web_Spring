[ REST ] : 웹 자원을 관리하는 아키텍처/구조
- HTTP 프로토콜(규약)을 이용한 클라이언트와 서버 간의 일관성있는 통신 구성ㅅ
[ REST API ]
- REST 아키텍처에 따른 API
- REST 기반의 웹서비스 인터페이스 구축
- HTTP 요청을 통해 자원을 CRUD할 수 있도록 제공
[ RESTful API ]
- REST API가 아키텍처를 잘 사용하는지에 대한 개념
- 조건
    1. 일관성 : "/member/XXXX", "/board/XXXX"
    2. 적절한 HTTP Method : POST(등록), GET(조회), PUT(수정), DELETE(삭제)
    3. 무상태성 : 서버는 클라이언트의 정보를 저장하지 않는다.
    4. 응답 타입 : JSON, XML...
